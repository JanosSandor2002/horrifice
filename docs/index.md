# Welcome to Horrifice Documentation

Welcome to the official documentation for **Horrifice**, a 2D pixel art RPG in development.  
This site serves as a central hub for information about the game, its mechanics, and design notes.

**Horrifice** is inspired by classic titles like _Final Fantasy IV_ and _Mardek RPG_, but with a darker, horror-themed twist.  
Players will explore a mysterious village, uncover hidden secrets, and engage in strategic, turn-based combat where agility and skill selection matter.
